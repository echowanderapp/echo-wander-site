개인정보 처리방침

최종 업데이트: 2025년 4월 12일

1. 소개

EchoWander(이하 "당사")는 사용자의 개인정보를 보호하기 위해 최선을 다하고 있습니다. 본 개인정보 처리방침은 당사 모바일 애플리케이션 사용 시 사용자의 정보를 어떻게 수집, 사용, 보호하는지를 설명합니다.

2. 수집하는 정보

2.1 위치 정보

당사는 다음의 목적으로 사용자의 위치 데이터를 수집합니다:
- 주변 명소 표시
- 정확한 오디오 투어 제공
- 명소까지의 거리 계산
이 데이터는 사용자가 권한을 허용한 경우에만 수집되며, 오직 앱 기능 제공 목적으로만 사용됩니다.

2.2 오디오 데이터

당사는 다음과 같은 목적을 위해 오디오 사용 데이터를 수집합니다:
- 투어 재생 추적
- 오디오 품질 개선
- 앱 성능 모니터링
당사는 사용자의 음성이나 개인 오디오 데이터를 기록하거나 저장하지 않습니다.

2.3 기기 정보

당사는 다음과 같은 정보를 수집합니다:
- 기기 종류
- 운영체제
- 앱 버전
- 사용 통계
이 정보는 앱의 성능 및 사용자 경험 개선에 활용됩니다.

3. 수집한 정보의 이용

수집된 정보는 다음과 같은 목적으로 사용됩니다:
- 앱 기능 제공
- 사용자 경험 개선
- 기술적 문제 해결
- 신규 기능 개발
당사는 사용자의 개인정보를 제3자에게 판매하거나 공유하지 않습니다.

4. 데이터 저장 및 보안

당사는 사용자의 정보를 보호하기 위해 다음과 같은 보안 조치를 시행합니다:
- 암호화된 데이터 저장
- 안전한 전송
- 정기적인 보안 업데이트
- 접근 제어

5. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:
- 자신의 데이터에 접근할 권리
- 데이터 삭제 요청
- 데이터 수집 거부
- 설정 및 선호도 변경

6. 아동의 개인정보 보호

EchoWander는 모든 연령대의 사용자가 안전하게 사용할 수 있도록 가족 친화적으로 설계되었습니다.

7. 본 정책의 변경

본 개인정보 처리방침은 변경될 수 있으며, 변경 시 본 페이지에 새로운 정책을 게시하여 안내합니다.
